JavaScript Console Quiz


Description:
This is a simple quiz game made using JavaScript.  
All the questions run in the browser console using prompt() and alert().  
The quiz checks user answers, gives immediate feedback, and shows the final score.

Main Features:
- Uses an array to store questions and answers  
- Uses a for loop to ask questions one by one  
- Uses functions to keep code organized  
- Accepts user input with prompt()  
- Uses toLowerCase() and trim() for accurate checking  
- Tracks score and displays it at the end  


How to Run:
1. Open any browser  
2. Right click → Inspect → Console  
3. Paste the entire JavaScript code  
4. Press Enter  
5. Answer the questions through prompts
